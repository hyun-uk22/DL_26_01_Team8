# Changelog

## 2026-06-01

- 레퍼런스 포즈 추출 전 현재 미리보기 프레임의 감지 인원 수를 확인하도록 추가.
- 레퍼런스 영상에서 여러 사람이 감지되고 ROI crop이 없으면 포즈 추출을 중단하고, 분석할 사람만 드래그로 선택하도록 안내.
- ROI crop 영역 안에 여러 사람이 남아 있으면 더 좁게 영역을 다시 선택하도록 안내.
- ROI crop 드래그 중 선택 영역이 실시간으로 보이도록 반투명 사각형 오버레이 추가.
- 레퍼런스 영상 재생 중 구간 슬라이더를 조정하면 재생 시작점이 변경된 시작 프레임으로 즉시 동기화되도록 수정.
- 영상 내부 재생 버튼과 프로그레스 바가 마우스 움직임 시 나타나고, 움직임이 없으면 하단으로 자동 숨김 처리되도록 수정.
- 영상 내부 컨트롤 바 배경을 영상 배경에 가까운 어두운 톤으로 조정해 반투명에 가까운 시각 효과 적용.
- 레퍼런스 구간 재생 중 `포즈 추출`을 누르면 재생을 자동 중단하고 바로 포즈 추출을 진행하도록 수정.
- ROI crop 드래그 종료 지점이 영상 영역 밖으로 나가도 영상 경계로 보정해 영역이 저장되도록 수정.
- ROI crop 저장 완료 시 선택 좌표를 상태바에 표시해 영역 지정 여부를 확인할 수 있도록 수정.
- ROI crop 기반 다중 인물 처리 흐름을 `MAIN_APPLIED_TECHNIQUES.md`에 반영.
